# react-native-yl-event-emitter

Event emitter for YL Modules

## Installation

```sh
npm install react-native-yl-event-emitter
```

## Usage

```js
import { multiply } from 'react-native-yl-event-emitter';

// ...

const result = await multiply(3, 7);
```
